# 더미 명령어

BlockTavern에는 더미 명령어가 추가되어 플레이어가 여러 가짜 플레이어를 시뮬레이션하여 게임 테스트를 수행할 수 있습니다.

## 명령어 목록

| 명령어 | 설명 |
| --- | --- |
| /player \<name\> spawn | 더미 생성 |

| /player \<name\> kill | 더미 삭제 |
