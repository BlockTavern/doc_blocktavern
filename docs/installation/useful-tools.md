# 🛠️ 常用工具推荐

## 启动器工具

### 🚀 MultiMC / PolyMC
**功能**: 强大的第三方启动器
- ✅ 实例隔离管理
- ✅ 模组包一键安装
- ✅ 内存和Java管理
- ✅ 版本独立配置

**下载地址**:
- [MultiMC](https://multimc.org/)
- [PolyMC](https://polymc.org/)

### 🎯 ATLauncher
**功能**: 模组包专用启动器
- ✅ 海量模组包资源
- ✅ 一键安装体验
- ✅ 自动依赖解决
- ✅ 服务器连接管理

**下载地址**: [ATLauncher](https://atlauncher.com/)

## 安装和使用建议

### 📝 工具选择原则
1. **根据需求选择** - 不要安装用不到的工具
2. **性能优先** - 优先选择轻量级工具
3. **稳定性考虑** - 选择维护良好的项目
4. **兼容性检查** - 确认与当前版本兼容

### ⚠️ 注意事项
- 定期更新工具版本
- 备份重要配置文件
- 测试工具功能完整性
- 阅读工具使用文档
